# calculate-the-marks-of-the-students-by-c-programing.
This project involves the creating of a electrical bill of stationary shop in the C programming language. The purpose of the project is to provide an interface where shopkeeper  can control the bill of the shop through the C programming .In this phase, the main focus was to understand the problem and requirements of shop billing.
Manual billing in shops is time-consuming to human errors.
To solve this issue, a computerized billing system was needed that could generate accurate and fast bills.
During research, various existing billing systems were studied to understand their features such as product entry, total amount calculation, and display.
The main goal was to design a simple, user-friendly, and efficient billing system suitable for small shop owners.
